## Overview

CodeSync is a modern full-stack collaborative coding platform built for developers who want to code, execute, and collaborate in real time.

It enables teams to:

- Write code together in synchronized rooms  
- Execute code instantly  
- Collaborate using whiteboards  
- Communicate through team chat  
- Use AI-powered development assistance  

Designed to deliver a professional developer experience inspired by modern engineering collaboration platforms.
